# nodejs版本管理工具 volta

快速无缝地安装和运行任何 JS 工具！Volta 是用 Rust 构建的，并以一个活泼的静态二进制文件的形式提供。

[volta文档地址](volta.sh)

volta 支持为不同的项目配置不同的 node 版本

在package.json添加 如下配置 即可

```json
{
  "volta": {
    "node": "16.18.0"
  }
}
```
